# Skills

Ponto de extensão para [skills do Claude Code](https://docs.claude.com/en/docs/claude-code/skills)
deste projeto. Cada skill vive em seu próprio subdiretório com um `SKILL.md` descrevendo quando e
como o Claude Code deve usá-la.

Nenhuma skill vem pré-instalada pelo template — adicione aqui as que fizerem sentido para o seu
projeto conforme ele evoluir. Skills de terceiros também podem ser adicionadas via
`npx skills add <pacote>` (ver exemplo de uso em [`scripts/plugins.sh`](../../scripts/plugins.sh)
para o `agent-browser`).
